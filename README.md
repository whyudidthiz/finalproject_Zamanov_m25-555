# ValutaTrade Hub

**ValutaTrade Hub** — это консольная платформа для симуляции торговли фиатными и криптовалютами. Проект состоит из двух сервисов:
- **Core Service** — управление пользователями, портфелями, совершение сделок (покупка/продажа) и отображение курсов.
- **Parser Service** — сбор актуальных курсов валют из внешних API (CoinGecko и ExchangeRate-API) и кэширование их в JSON-файлах.
